# ARCHITECTURE OVERLAY REPORT

## Overview
The Architecture Overlay provides a high-level, visual summary of the internal logic and data flow of the OPTIMUS application. It is designed to quickly communicate the technical sophistication of the system to technical judges.

## Implementation Details
1. **System Flow Visualization**: A linear, vertically-oriented pipeline using lucide-react icons and CSS gradients. It traces data from external sources (Gmail, Calendar) through the Discovery Agents, Risk Engine, and ultimately to the Execution layer.
2. **Tech Stack Rollup**: A dedicated side panel highlighting the modern stack: Next.js 16, React 19, Supabase, Gemini 2.5 Flash, and Framer Motion.
3. **Access**: The overlay is fully integrated into the Judge Quick Actions menu and the Tour End Screen, allowing judges to pivot from exploring UI features to understanding the backend architecture with a single click.

## Impact
By abstracting the complexity of the codebase into a clean, cyberpunk-inspired visual diagram, OPTIMUS successfully appeals to engineering judges who prioritize system design alongside visual polish.
